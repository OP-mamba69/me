# 2D Platformer Game (Python + Pygame)

A Mario-style class 11 project showcasing:

- Gravity and jumping
- Collision detection (player/platform, enemy/platform)
- Enemy and health system
- OOP design with `Player`, `Enemy`, and `Level` classes
- Simple game state management (`menu`, `playing`, `game_over`, `win`)

## Run

```bash
python3 -m pip install -r requirements.txt
python3 main.py
```

## Controls

- Move: `A/D` or `←/→`
- Jump: `W` / `↑` / `Space`
- Start: `Enter`
- Retry after lose/win: `R`
- Quit: `Esc`

## Project structure

- `main.py` — complete game logic and classes
- `requirements.txt` — dependencies
